# Data Definition Language (DDL)

[Video Link](https://youtu.be/ltP6lK_TCpE)

DDL is used to create and modify database objects like tables, indices, views etc.

Few DDL commands in SQL are listed below:

- CREATE
- ALTER
- RENAME
- TRUNCATE
- DELETE
